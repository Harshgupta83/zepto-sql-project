Zepto SQL Data Analysis Project

Overview

This project focuses on analyzing Zepto's product dataset using SQL. The objective was to clean the data, explore key metrics, and generate business insights that can support data-driven decision-making.

Project Objectives

- Perform data cleaning and preprocessing
- Explore product and inventory data
- Analyze pricing and discount patterns
- Generate business insights through SQL queries
- Practice real-world SQL problem-solving

Skills Used

- SQL
- Data Cleaning
- Data Exploration
- Aggregate Functions
- Joins
- Subqueries
- Business Analytics

Analysis Performed

Data Cleaning

- Identified and handled missing or inconsistent values
- Standardized data for accurate analysis

Data Exploration

- Examined product categories and inventory status
- Analyzed pricing, discounts, and stock availability
- Explored dataset structure and key metrics

Business Insights

- Category-wise product distribution
- Discount and pricing trends
- Inventory analysis
- Product availability insights
- Business-focused query solutions

Project Files

- "zepto_analysis.sql" – Contains all SQL queries used for data cleaning, exploration, and business insights.
- "zepto_dataset.csv" – Dataset used for analysis.

Key Takeaway

This project helped strengthen practical SQL skills by working with a real-world e-commerce dataset and extracting meaningful insights from raw data.
